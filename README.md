# 🧩 NEPv Interactive Visualizer

This website is an educational tool designed to visualize **Nonlinear Eigenvalue Problems (NEPv)** where the operator matrix depends dynamically on its own eigenvector.

## 🚀 Features
* **Graphical Sandbox:** Drag an input vector manually on a 2D grid to observe how the corresponding matrix field shifts its shapes in real time.
* **Algebraic Tracking:** Compare optimization path trajectories between **Self-Consistent Field (SCF)** loops and local **Newton-flavor** root-finding techniques.

## 🛠️ How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Launch the framework: `streamlit run app.py`
